# screeps
my screeps game code
# Install
> npm install

create file .secret.json in root 
```
{
    "main": {
        "token": "",
        "protocol": "https",
        "hostname": "screeps.com",
        "port": 443,
        "path": "/",
        "branch": "default"
    },
    "local": {
        "copyPath": ""
    }
}
```

write token and copyPath to your own token and copyPath
